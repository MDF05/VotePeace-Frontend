# VotePeace Frontend

![Frontend Status](https://img.shields.io/badge/status-active-success.svg)
![React Version](https://img.shields.io/badge/react-19.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-blue.svg)

**VotePeace-Frontend** is the modern, responsive user interface for the VotePeace election platform. Built with **React 19** and **Material UI (v7)**, it delivers a premium, real-time voting experience with advanced analytics and seamless interactions.

## 🚀 Frontend Overview

The frontend is a Single Page Application (SPA) powered by **React Router v7** and **Vite**, focusing on:
-   **Voter Experience**: Intuitive voting flow with real-time feedback.
-   **Admin Dashboard**: Comprehensive analytics, campaign management, and user oversight.
-   **Real-time Data**: Live election results visualization using Recharts.
-   **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices.

## ✨ Features

-   **Modern Tech Stack**: React 19, React Router v7, Vite.
-   **UI System**: Material UI v7 combined with Tailwind CSS v4 for flexible styling.
-   **State Management**: Redux Toolkit for global state and efficient data handling.
-   **Form Handling**: React Hook Form with Zod validation.
-   **Data Visualization**: Interactive charts (Bar, Pie) using Recharts.
-   **Security**: Protected routes, JWT handling, and secure API integration.

## 🛠 Tech Stack

-   **Framework**: [React 19](https://react.dev/) + [React Router v7](https://reactrouter.com/)
-   **Build Tool**: [Vite](https://vitejs.dev/)
-   **Styling**: [Material UI v7](https://mui.com/), [Tailwind CSS v4](https://tailwindcss.com/)
-   **State Management**: [Redux Toolkit](https://redux-toolkit.js.org/)
-   **Forms**: [React Hook Form](https://react-hook-form.com/) + [Zod](https://zod.dev/)
-   **Charts**: [Recharts](https://recharts.org/)
-   **HTTP Client**: [Axios](https://axios-http.com/)

## 📦 Installation

Prerequisites:
-   [Node.js 20+](https://nodejs.org/)
-   npm or yarn

```bash
# Clone the repository
git clone https://github.com/MDF05/VotePeace-Frontend.git

# Navigate to the project directory
cd VotePeace-Frontend

# Install dependencies
npm install
```

## 🚀 Development Workflow

To start the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:5173`.

### Type Checking
To run TypeScript compiler checks:
```bash
npm run typecheck
```

## 🏗 Build & Deployment

To build for production:
```bash
npm run build
```

To preview the production build locally:
```bash
npm run start
```

For deployment guidelines, see [DEPLOYMENT.md](./DEPLOYMENT.md).
